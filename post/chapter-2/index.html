Project II: Fama French Three Factor Portfolio Construction

August 3, 2020

Background:

Few months ago when I started my undergrad thesis, I need to get SMB and HML factors for Fama-French Three Factors Model (my thesis is about asset pricing model). Since I use sample of stock outside US (niche sample), I need to calculate both factors on my own instead of getting it from Kenneth French’s website. I wasn’t able to code back then, and almost crazy enough to use excel. Fortunately, I was able to get MSCI index that provide Small-Cap, Large-Cap, Value, and Growth indices of my sample’s stock market. But most of the time, it maybe the case that our sample doesn’t match with the constituents of index provided by MSCI. Hence, it might be better to construct the factors on our own. This motivation led me to start this project. I hope it can help student(s) to construct Fama French portfolios and get SMB & HML factors with any sample.

I won’t discuss thoroughly behind the theory of Fama-French Models, but the main hypotheses are small-cap stocks tends to outperform large-cap stocks and value stocks (high book-to-market) tends to outperform glamour or growth stocks (low book-to-market). Thus, we can construct six portfolios to test the hypotheses: Small-Growth, Small-Neutral, Small-Value, Big-Growth, Big-Neutral, and Big-Value.

For more details on Fama French Three Factors, read these two journals; “The Cross-Section of Expected Stock Returns” & “Common risk factors in the returns on stocks and bonds”.

Input

  • List of companies in xlsx format with Company as column name and comp_list as file name
  • Market-cap and Price to Book or Book to Market for each company in xlsx format. Follow the format of example raw data, if you extract both features from Bloomberg Terminal it should be in the similar format. If your raw data is already in rows x [[‘market_cap’,‘b_m’]] with date index format, just skip the first step (organize raw data).

Note. I wasn’t able to find free online resources for market-cap and book-to-market data. For most finance or business student, your school or uni should provide you access for Bloomberg Terminal.

Process

  • Wrote in Python
  • The code utilize Pandas DataFrame for portfolio construction
  • SMB is defined as 1/3(Small Growth + Small Neutral + Small Value) - 1/3(Big Growth + Big Neutral + Big Value)
  • HML is defined as 1/2(Small Value + Big Value) - 1/2(Small Growth + Big Growth)

Output

  • Market or index return
  • SMB factor
  • HML factor
  • Small; -growth, -neutral, -value portfolio return
  • Big; -growth, -neutral, -value portfolio return

Note. All return is value-weighted return.

Simple Analysis

I use example of 46 companies listed on Kompas 100 in the code. Figure below visualize the cummulative return of all 6 portfolios constructed.

As we can see above, Small Value portfolio has the highest return in most of the time period. Although in the early period, it has the smallest return.

It seems that visualization has a nice way to confirm Fama-French hypotheses but it wasn’t enough. Thus, statistical check is required for more thorough anlysis. Table below shows the regression of each portfolio on Market Return, SMB and HML

The regression results clearly shows that:

(1) All large-cap portfolio return is significant and negatively related with SMB factor.

(2) All small-cap portfolio return is significant and positively related with SMB factor.

(3) All value portfolio return is significant and positively related with HML factor.

(4) All growth portfolio return is significant and negatively related with HML factor.

Some features may has weak significance (10% p-val) due to small observation. Nevertheless, we can confirm Fama-French hypotheses.
